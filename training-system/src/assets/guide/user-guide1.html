Введение

В системе присутствует две роли: учитель и ученик. Учитель может составлять обучающие задания и следить за успеваемостью своих учеников, которые в свою очередь, решают поставленные задачи при помощи графических примитивов.

Авторизация

Для регистрации в системе вам необходимо ввести ФИО, логин, пароль, а также выбрать роль: учитель или ученик. По нажатию на кнопку "Создать аккаунт" ваш аккаунт будет создан.

Для входа в систему вам необходимо ввести логин и пароль и нажать кнопку "Войти".

Режим "Учитель"

После авторизации под ролью "Учитель", интерфейс будет подстроен под роль. Перед вами будет страница с созданными заданиями.

Создание задания

Чтобы создать новое задание, необходимо нажать на кнопку "Доабвить". После этого будет открыта страница создания задания.

Страница разделена на 3 блока: блок с объектами в виде графических примитивов, блок с названием и описания задания, блок с игровым полем. Вам необходимо ввести название задание и его описание, а также настроить игровое поле (ИП).

Ручная настройка ИП

Для ручной настройки ИП вам необходимо вручную расставить обекты на поле. Для расположения объекта на игровом поле, вы должны перетащить графический примитив на соответствующую клетку игрового поля. Также можно очистить поле по нажатию на кнопку "Очистить". Чтобы проверить созданное задание на корректность, нажмите на кнопку "Проверить".

Автоматическая генерация ИП

Для автоматической генерации ИП, необходимо нажать на кнопку "Сгенерировать". В открывшемся модальном окне вы сможете выбрать количество объектов определенного типа, которые должны присутсвовать на поле. После этого нажмите на кнопку "Применить". Игровое поле заполнится случайным образом в соответствии с настройками генерации.

Сохранение и публикация задания

После создания задания вам доступны две опции: сохранить без публикации, либо опубликовать задание. Для сохранения без публикации нажмите на кнопку "Сохранить", в таком случае вы сможете в дальнейшем изменять это задание, а ваши ученики видеть его не будут.

Для публикации задания нажмите на кнопку "Опубликовать". В этом случае вы больше не сможете изменять задание, а ваши ученики смогут приступить к его решению.

Управление журналом

Для перехода к просмотру журнала, нажмите на кнопку "Журнал" в навигационном меню. Перед вами откроется страница со списком ваших учеников.

Добавление ученика

Чтобы добавить ученика в свою группу нажмите на вкладку "Добавить" на странице журнала. После этого на странице будет отображаться список со всеми учениками в системе. Для того, чтобы добавить ученика в группу, нажмите кнопку "Добавить" напротив ФИО необходимого ученика.

Удаление ученика

Чтобы удалить ученика из своей группы нажмите на кнопку "Удалить" напротив ученика в списке вашей группы.

Просмотр статистики

Для просмотра статистики определенного ученика из вашей группы нажмите на кнопку "Статистика" напротив этого ученика. После этого откроется модальное окно с оценками ученика за каждое выполненное задание.

Для просмотра статистики по всему журналу нажмите на кнопку "Статистика" над списком учеников. После этого откроется модальное окно с средним баллом каждого ученика в группе.

Проверка решения

После того как ученик прислал решение его можно проверить. Для этого перейдите на страницу "Задания" и нажмите на карточку необходимого задания. Откроется страница првоерки задания, разделенная на 2 блока: блок с алгоритмом ученика, блок с игровым полем.

Выберите ученика из выпадающего списка над блоком команд. Вы увидите алгоритм, составленный учеником. Для его проверки нажмите на кнопку "Воспроизвести". Далее выберите оценку и нажмите на кнопку "Оценить"

Режим "Ученик"

После авторизации под ролью "Ученик", интерфейс будет подстроен под роль. Перед вами будет страница с назначенными заданиями.

Создание решения

Чтобы решить задание нажмите на карточку с заданием из списка. Вам откроется страница с составлением решения.

Составление алгоритма

Страница разделена на 3 блока: список команд, созданный алгоритм и игровое поле. Вы должны, перетаскивая команды из списка команд в блок с алгоритмом, составить алгоритм решения задания. Вы можете очистить созданный алгоритм нажав на кнопку "Очистить" под алгоритмом.

Визуализация алгоритма

У вас есть возможность воспроизвести созданный алгоритм, нажав на кнопку «Воспроизвести» под игровым полем.

Сохранение и отправка решения на проверку

После создания задания вам доступны две опции: сохранить без отправки, либо отправить решение на проверку. Для сохранения без отправки нажмите на кнопку "Сохранить", в таком случае вы сможете в дальнейшем изменять этот алгоритм, а учитель не сможет увидеть вашего решения

Для отправки решения на проверку нажмите на кнопку "Отправить". В этом случае вы больше не сможете изменять решение, а учитель сможет приступить к проверку вашего решения.

Просмотр оценок

Для перехода к просмотру журнала, нажмите на кнопку "Оценки" в навигационном меню. На открытой странице вы увидите список всех решенных вами задач, которые уже проверены учителем. На основе ваших оценок строится гистограмма.

Завершение работы

Для выхода из системы нажмите на кнопку "Выход" в навигационном меню.
